# 💫 About Me:
Sou desenvolvedor back-end, focado na construção de APIs e sistemas que lidam bem com dados e escala. Trabalho principalmente com Node.js, Express e bancos de dados, desenvolvendo aplicações que organizam e processam informações de forma eficiente.<br><br>Também tenho interesse em segurança e criptografia, aplicando conceitos como hashing e autenticação para proteger dados e usuários dentro das aplicações.<br><br>Sou estudante de Análise e Desenvolvimento de Sistemas e estou sempre explorando novas tecnologias, padrões de arquitetura e boas práticas de desenvolvimento. Gosto de entender como os sistemas funcionam por trás das interfaces e de resolver problemas usando lógica e código.<br><br>
